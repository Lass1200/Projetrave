# Projetrave
Veuillez trouver ci-joint les éléments de rendu du projet RAVE :

GitHub : https://github.com/lass1200/Projetrave
Expo : https://expo.dev/accounts/lassanasamassa2/projects/rave-app
Pour tester l'application, il faudra :

Lancer npx expo start depuis le projet
Scanner le QR code avec Expo Go
Lancer le serveur Python python3 server.py sur le même réseau


**RAVE** est une application mobile React Native permettant d'enregistrer des clips audio, de les envoyer à un serveur Python qui les transforme via un 
modèle d'intelligence artificielle (VAE), et d'écouter le résultat. Par exemple, ta voix peut être transformée
en son de jazz, darbouka, chats ou chiens. L'app est composée de 3 écrans : connexion au serveur, enregistrement/gestion des clips, et transformation audio.
